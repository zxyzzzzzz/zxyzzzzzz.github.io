关于GIT的一些指令 git add . （注意有个点号）将当前目录的文件提交到GIT的本地仓库中中 git commit -m “《提交的备注在这里写》” 将添加到本地仓库中的代码合并 并设置提交备注 git push -u og master 将本地仓库代码推送到远端master分支上 （og是我最新设置的博客网址别名 ) ps：如果git推送不成功可以考虑拖过去但可能会出错
