hugo安装博客的笔记
@ zxy · Oct 2, 2019 · 1 min read · Update at Oct 2, 2019

1.以后应该会用到的 hugo new post/“要写的博客的名字用.md后缀结尾” //新建文章 hugo server -t “主题名称”–buildDrafts //在本地浏览器查看网页 git remote add origin https://github.com/zxyzzzzzz/zxyzzzzzz.github.io.git git push -u ori master //这两条是推送到服务器上

2重装的时候会用吧 hugo new site “站点名称”//生成一个新站点 hugo –theme=“主题名称” –baseUrl=“https://“站点名称”.github.io/” –buildDrafts

上传流程 首先 hugo –theme=dream –baseUrl=“https://zxyzzzzzz.github.io/" –buildDrafts 此时public文件夹创立了 我们需要进入 cd public 进入后 git init (初始化) 然后git add .（更新） 接着 git commit -m “第n次提交”

git remote add origin https://github.com/zxyzzzzzz/zxyzzzzzz.github.io.git 此时就可以用上传到服务器上了

git push -u origin master

二次上交有一个比较简单适合小白的方法就是把public删掉然后重新弄一个出来直接拖到库里覆盖

hugo –theme=“主题名称” –baseUrl=“https://“站点名称”.github.io/” –buildDrafts
